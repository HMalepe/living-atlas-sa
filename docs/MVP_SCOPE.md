# MVP Scope — Living Atlas SA

## Geographic focus

Johannesburg area only. No nationwide coverage in MVP.

### Minimum data targets

- 20 road segments
- 15 major intersections
- 20 road relationships
- 30 timeline events
- 20 sources
- 10 community-memory placeholders (clearly marked sample/unpublished)
- 30 quiz questions

## Sky MVP — celestial objects

Sun, Moon, Mercury, Venus, Mars, Jupiter, Saturn, Southern Cross, Orion, Aries, Sirius, Alpha Centauri, Magellanic Clouds, Milky Way centre direction.

### Features

- Live location-aware sky page
- Moon phase display
- Rise/set information
- Planet finder
- Constellation tracing prototype (Aries, Southern Cross)
- South African seasonal sky guide
- 30 sky quiz questions

## Unified MVP

- User accounts (Supabase Auth)
- Passport with saved entities and progress
- One flagship journey: **Johannesburg: Roads Below, Sky Above**
- Global search with match explanations
- Admin portal with role-based access
- Source display and confidence badges
- Mobile-first interface; desktop research experience

## Out of scope for MVP

- Nationwide road network
- Payments and subscriptions (schema placeholders only)
- Native AR (architecture only)
- Full multilingual UI (architecture only)
- AI-published historical claims

## Definition of done

See master build prompt §33. MVP is complete only when all listed criteria pass — including tests, RLS verification, and no invented historical facts in production.
