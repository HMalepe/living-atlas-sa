# User Journeys — Living Atlas SA

## Primary user types

1. Curious South African residents
2. School learners and teachers
3. Tourists and remote explorers
4. Road and infrastructure enthusiasts
5. Astronomy beginners and amateur astronomers
6. Historians, urban planners, researchers
7. Community elders and storytellers
8. Municipal and tourism partners

Physical travel adds value but must not lock out users with mobility, safety, location, or cost limitations.

---

## Journey 1: First visit — horizon discovery

1. Land on homepage with ground/sky horizon
2. Drag or tap to shift between Ground and Sky
3. Read headline and trust messaging
4. Choose "Explore the Ground" or "Explore the Sky"
5. Return later via featured discovery or passport

**Success:** User understands the dual-world concept within 30 seconds.

---

## Journey 2: Road curiosity — "What is this road?"

1. Open Ground map (Johannesburg)
2. Tap a highlighted road (e.g. M1)
3. Preview card shows name, route number, one-line purpose
4. Open full road page: identity, anatomy, timeline, sources
5. Notice "Things You Might Never Notice"
6. Save to passport

**Success:** User learns something non-obvious with visible source confidence.

---

## Journey 3: Parallel roads comparison

1. From M1 page, see relationship to parallel roads
2. Open "Why do these roads run parallel?"
3. Compare age, function, naming, communities served
4. Complete compare challenge for XP

**Success:** User understands evidence-based explanations, including disputed or unknown areas.

---

## Journey 4: Tonight's sky

1. Open Sky → Live
2. Grant location (or use approximate Johannesburg)
3. See visible planets, Moon phase, rise/set times
4. Tap Jupiter → learn why it looks like a bright star
5. Trace Southern Cross constellation

**Success:** User identifies at least one object and understands visibility requirements.

---

## Journey 5: Flagship journey — Johannesburg: Roads Below, Sky Above

1. Start journey from `/journeys/johannesburg-roads-below-sky-above`
2. Passenger/parked mode selection with safety messaging
3. Three road stops + intersection + naming story + community memory
4. Safe sky-observation point with Moon/constellation discovery
5. Final comparison challenge
6. Post-journey review and passport stamp

**Success:** Journey completable without unsafe driving interaction.

---

## Journey 6: Community contribution

1. Open `/contribute` from road or sky object page
2. Submit oral history, photo, or nickname with consent
3. Track moderation status
4. If published, see contribution labelled as community memory

**Success:** Submission never appears as verified official fact without review.

---

## Journey 7: Researcher editorial workflow

1. Sign in → `/admin`
2. Create draft road entity, attach geometry, claims, sources
3. Flag missing citations in dashboard
4. Submit for review → moderator publishes
5. Audit history preserved

**Success:** No claim published without source attachment option and confidence label.
